# 机器视觉课程实验项目（2025）

本仓库汇总《机器视觉》课程 4 个实验的代码/数据/报告，覆盖 **传统图像处理**、**经典几何检测**、**机器学习/深度学习识别** 与 **YOLO 目标检测** 等完整链路。

作者：折浩宇 学号：2023217896

- 实验一：图像滤波（手写卷积 + Sobel 边缘 + 颜色直方图 + 纹理特征 LBP/GLCM，保存 `.npy` 特征）
- 实验二：车道线检测（HSV/HLS 阈值分割 + 形态学 + Canny + ROI + 概率霍夫直线检测）
- 实验三：学号识别（MNIST 上训练 SVM 与 CNN，并在真实学号照片上完成数字分割与识别）
- 实验四：校园共享单车检测（YOLO 迁移学习：COCO 基线 → 自建数据集微调，三类：哈啰单车/电动车/汽车）
